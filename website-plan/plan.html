W05 Website Plan - WDD 231 - Cliff Cummings

1. Site Name

sunburst-design.com

This has been my own company website for more than 25 years. I ran my own company from 1995 -2020, when I merged my company with Paradigm Works (paradigm-works.com). The Sunburst Design site name is known by engineers worldwide. A quick Google search of "cliff cummings sunburst design" shows just how widely the name is known and recognized.

I have already created and applied a sunburst-logo favicon.

2. Site Purpose

The site name is probably most famous for the conference and technical papers that I have presented over the past 31 years. This site will provide a much better organization of those papers than what is currently found on the www.sunburst-design.com/papers web page.

3. Scenarios

Three scenario questions are shown that reflect on interests of the target audience users.

  1. Where can I find Cliff's good technical content related to Verilog design?
  2. Where can I find Cliff's good technical content related to SystemVerilog design?
  3. Where can I find Cliff's good technical content related to UVM (Universal Verification Methodology) verification?

4. Color Schema

At least two colors in the document will be:

:root {
--header_color1: #0D1B2A;
--sunburst_red: #CC0000;
}

The header_color1 color will be used for h2 and h3 headers. The sunburst_red color is used for the logo h1 text.

5. Color Schema Use in Document

This site will use "--sunburst-red" for the <h1> text beneath the sunburst-design logo and the Page Header title. It matches the color of the logo.

This site will use "--header_color1: #0D1B2A" for other headers in the document.

Additional schema colors will be used as deemed appropriate.

6. Typography Named and Used

This document will use: fonts.googleapis.com/css2?family=Roboto:wght@100;400&display=swap"

This document will also use: fonts.googleapis.com/css2?family=Arimo:ital,wght@0,400..700;1,400..700&display=swap"

These fonts are the general fonts used throughout the document.

Additional fonts might be used as deemed appropriate.
